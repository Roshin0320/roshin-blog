关于博客

关于本站 #

这是一个兼具博客文章、知识管理、文档查找的个人网站，主要内容是Web前端技术。

网站主题 #

本站使用Vuepress (opens new window)生成，主题改编自Vdoing (opens new window)，这是一款简洁高效的知识管理&博客主题。感谢大佬的开源分享！想了解更多请点击详情 (opens new window)。

使用到的插件 #

本博客基于vuepress的二次开发，在个性化定制的过程中使用到了以下插件：

  • vuepress-plugin-element-tabs (opens new window) - 选项卡

  • vuepress-plugin-baidu-autopush (opens new window) - 百度站点自动推送

  • vuepress-plugin-sitemap (opens new window) - sitemaps 生成器

  • vuepress-plugin-thirdparty-search (opens new window) - 可以添加第三方搜索链接的搜索框

  • vuepress-plugin-one-click-copy (opens new window) - 代码块一键复制

  • vuepress-plugin-demo-block (opens new window) - 代码演示块

  • vuepress-plugin-zooming (opens new window) - 图片点击缩放

  • vuepress-plugin-baidu-tongji (opens new window) - 百度统计

  • vuepress-plugin-comment (opens new window) - 第三方评论区
